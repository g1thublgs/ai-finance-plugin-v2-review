# 现有系统架构与目录说明

```text
Chrome 插件前端
  -> 上传附件 / 提取页面
  -> Node 后端
      -> OCR 服务
      -> 场景识别
      -> 场景预填归集
      -> 场景规则审核
      -> SQLite 记录调试过程
  -> 插件工作台展示结果
  -> 按场景回填网页
```

## 源码目录

```text
新插件源码/
|-- 预填预审事中审核一体化插件/   Chrome 插件前端
|-- backend/                    Node 后端工程
|-- poppler/                    PDF/OFD 处理组件
`-- tools/                      SQLiteStudio 等辅助工具
```

## 场景目录

```text
backend/server/src/domain/scenarios/
|-- smart/       智能入口
|-- other/       其他事项报销，保留展示和借鉴
|-- meeting/     会议费，由对应地市开发
|-- training/    培训费，由对应地市开发
|-- reception/   公务接待费，由对应地市开发
`-- shared/      场景公共工具
```
