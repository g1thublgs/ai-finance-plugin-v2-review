# 本地开发环境部署说明

## 需要环境

| 环境 | 用途 | 说明 |
|---|---|---|
| Node.js | 启动后端服务 | 建议 Node 18 或以上；分发包已带 `node_modules`，通常不需要重新安装依赖。 |
| Chrome 或 Edge | 加载插件 | 使用“开发者模式”加载插件文件夹。 |
| Python 3 | Python 规则 | 如要运行 `services/rules/python/<scenario>/rule_*.py`，本机需可执行 `python`、`python3` 或 `py -3`。 |
| Poppler | PDF 渲染 | 已随源码保留。 |

## 启动后端

```bat
cd 新插件源码\backend
npm run check
npm start
```

或双击：

```text
新插件源码\backend\检查后端语法.bat
新插件源码\backend\启动后端服务.bat
```

默认地址：

```text
http://127.0.0.1:3000
```

## 加载插件

1. 打开 Chrome/Edge 扩展程序页面。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择 `新插件源码\预填预审事中审核一体化插件`。
5. 确认后端已经启动。
