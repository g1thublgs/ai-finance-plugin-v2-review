# AI财务预填预审审核助手操作说明

本插件为预填预审插件和事中审核插件的一体化版本，建议后续测试优先加载本目录。

## 一、安装

1. 启动后端服务：`系统\插件\场景化插件重构版\backend\启动后端服务.bat`
2. 打开 Chrome：`chrome://extensions/`
3. 开启“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择目录：`系统\插件\场景化插件重构版\预填预审事中审核一体化插件`

插件默认访问后端：`http://150.88.16.204:3000`。

## 二、两种工作模式

### 1. 上传附件预填预审

适合报销人员填单前使用。

1. 打开财务系统对应报销页面。
2. 点击插件图标打开浮窗。
3. 选择“上传附件预填预审”。
4. 选择审批单、机票、火车票、住宿清单、发票等附件。
5. 点击“上传识别并进入工作台”。
6. 插件完成 OCR、归集和预审后，进入工作台展示预填汇总、差旅行程、报销明细和 OCR 调试数据。
7. 核对无误后，可点击“一键填写至网页”。

该模式会显示“一键填写至网页”按钮。

### 2. 提取当前页面审核

适合审核人员查看已经填写好的报销单。

1. 打开财务系统中待审核的报销页面。
2. 点击插件图标打开浮窗。
3. 选择“提取当前页面审核”。
4. 点击“提取当前页面并审核”。
5. 插件会提取页面已填写的人员、金额、付款信息、经济科目和已上传附件。
6. 插件下载页面附件并上传后端 OCR，再调用指标审核接口。
7. 工作台重点展示“指标命中”和 OCR 调试结果。

该模式不会显示“一键填写至网页”按钮，避免审核阶段误修改页面。

## 三、数据来源规则

```text
上传附件模式：
本地附件 -> 后端OCR -> 场景归集 -> 指标预审 -> 前端展示 -> 可一键填写

页面提取模式：
当前页面数据 + 页面已上传附件 -> 后端OCR -> 指标审核 -> 前端展示 -> 不允许一键填写
```

## 四、当前支持场景

```text
差旅费：支持上传预填预审、页面提取审核、差旅规则指标命中。
其他事项：支持上传识别生成预填汇总、页面提取审核和金额/预算基础审核。
会议费、培训费、公务接待费：目录和接口已预留，后续按场景补充 OCR、归集和规则。
```

## 五、注意事项

- 页面提取模式依赖财务系统附件表中的文件流水号，附件下载地址当前为 `http://86.16.27.237:9010/cw/net/hitina/biz/nk/fjsc/FjscUnite.downloadFile.svc?fjLsh=`。
- 若内网文件服务器地址变化，需要同步修改 `popup.js` 中的 `FILE_SERVER_URL`。
- 上传附件模式仍保留原有预填预审交互和一键填写逻辑。
- 页面提取模式只做审核展示，不回填、不保存、不修改财务页面。
